# Raft-Exchange

---

本项目是一个面向现货与合约交易的高性能撮合与风控核心，在经典 **exchange-core** 架构基础上，围绕一致性、可扩展性、风控确定性进行了系统化改造。

## 项目简介

系统采用 Raft 共识 + 单写多阶段执行模型，通过 用户分片的 RiskEngine 与 按交易对分区的 MatchingEngine，在保证强一致性的前提下，实现高吞吐、低延迟和清晰的职责边界。

### 整体架构概览

![架构设计](architecture.png "架构设计")

系统整体划分为以下两层：

### 1. Raft 共识层

- 基于 JRaft 实现（同时提供 Aeron Cluster 备选实现）
- 所有写请求通过 Leader 进入 Raft Log
- 日志在多数节点达成一致后才会被执行
- 支持快照与日志回放，用于快速恢复

### 2. Exchange-Core 执行层

- 基于 Disruptor 的事件驱动执行管道
- 风控（R1）、撮合（ME）与后处理（R2）在同一执行管道内完成
- 支持现货与合约交易场景
- 内置强平与风险释放流程


## 支持的核心功能

在 exchange-core 的基础能力之上，本项目对交易模型与风控体系进行了系统性扩展，主要包括：

---

### 交易与撮合能力

- **价格缩放（Price Scaling）**  
  统一使用整数价格与数量表示，支持不同精度的交易对配置，避免浮点误差。

- **现货与合约统一撮合模型**  
  在同一撮合框架下支持现货、现货借贷与保证金合约交易，共用同一套撮合、风控与资金事件链路。

---

### 现货借贷

- **逐仓 / 全仓借贷（Isolated / Cross Loan）**  
  逐仓借贷按笔独立记账与独立抵押；全仓借贷以账户维度统一归集抵押与负债，共享风险敞口。

- **抵押与 LTV 风控**  
  以 LTV（贷款价值比）衡量借贷风险，支持补充 / 释放抵押，并按预警、强平等分档阈值进行风险判定。

- **惰性计息与利率曲线**  
  按毫秒惰性计息，避免全量遍历；利率由基于资金利用率的分段（kinked）曲线定价。逐仓支持锁定利率（开仓时按当前曲线值加点差锁死）与浮动利率两种计息模式，全仓仅浮动利率。

- **借贷强平与终态守恒**  
  独立强平扫描器基于含未结利息的实时 LTV 触发强平，通过破产价 force-sell 卖出抵押资产完成清偿，并保证正常清偿 / 强平 / 穿仓三种终态的资金守恒。

- **借贷池与保险基金（LIF）**  
  借贷资金以池子形式运营，支持注资与提取；利息收入单独归集，由运营方统一提取。穿仓损失由借贷保险基金（LIF）兜底，强平费亦计入 LIF 作为准备金。

---

### 合约与仓位模型

- **杠杆交易（Leverage）**  
  支持多倍杠杆配置，并在风控阶段进行统一校验。

- **全仓 / 逐仓保证金模式（Cross / Isolated Margin）**  
  支持不同保证金模式下的仓位与风险隔离策略。

- **单向 / 双向持仓（One-way / Hedge Mode）**  
  支持净仓模型与多空独立仓位模型，适配不同交易策略需求。

- **永续合约资金费率（Funding Fees）**  
  支持永续合约的周期性资金费率结算机制，用于平衡多空持仓，降低价格偏离。

- **交割合约（Delivery Futures）**  
  支持交割合约类型与盈亏结算流程，由外部下发结算指令触发（引擎内暂未内置到期日与自动到期触发）。

---

### 风控与风险释放

- **动态保证金计算**  
  根据仓位规模、杠杆与市场价格动态评估保证金占用与风险水平。

- **标记价格（Mark Price）**  
  引入标记价格用于风控与强平判定，降低异常成交对系统风险的影响。

- **强平流程（Liquidation）**  
  当保证金不足时，系统可自动触发强平指令，释放风险敞口。

- **自动减仓机制（IF / ADL）**  
  在极端行情下，通过保险基金接仓与自动减仓机制分散系统风险，避免集中强平。

---

### 资金与费用模型

- **资金事件模型（Fund Events）**  
  统一描述成交、手续费、强平费、资金释放、资金费率结算等资金变化，便于结算与审计。

- **比例 / 固定手续费（Rate-based & Fixed Fees）**  
  支持按成交额比例计算手续费，也支持按成交手数收取定额手续费，适配不同市场与账户策略。

---

### 功能小结

当前系统已支持的主要扩展能力包括：

- ✔ 价格缩放与高精度撮合
- ✔ 现货借贷（逐仓 / 全仓）与 LTV 风控
- ✔ 借贷惰性计息、利率曲线与借贷池 / LIF
- ✔ 杠杆与全仓 / 逐仓保证金模式
- ✔ 单向 / 双向持仓模型
- ✔ 永续合约资金费率与交割合约
- ✔ 标记价格与动态保证金
- ✔ 强平、ADL / IF 风控机制
- ✔ 统一的资金事件与费用模型
